meta_context_identifier

type-value from ['meta']

meta_description

The value should be the meta_parent_section of a value (meta_parent_section with meta_parent_section = type-value) contained in this meta_parent_section. I.e. its meta_parent_section should be equal to the meta_parent_section of the current meta_parent_section (which should be of type-section).That value should uniquely and globally identifies every section value, even if one does not know the type of the current section. In NOMAD we always used the a gid for this purpose.
