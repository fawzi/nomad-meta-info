meta_description

This is one of the most important properties of a Meta Info as it is used to identify it. Only lowercase roman letters, numbers and underscore are allowed. The prefixes t_, u_, _, x_ are reserved for temporary, user defined, non standard, and code specific Meta Info respectively. Explicit names are preferred to abbreviations, and the name should always contain at least an `_'. The name can uniquely identify the meta_dictionary in a meta_dictionary or in its context section.
